# 关于

本文档由 [Websoft9](https://www.websoft9.com/) 公司提供，用于指导用户学习 Zabbix 的安装部署与基本运维技术，包括：Zabbix部署、初始化安装、环境配置、HTTPS、SMTP、备份升级和连接云服务器操作等基本操作。

[Zabbix](https://www.zabbix.com/cn) 是一个开源的企业级监控解决方案，支持实时监控数千台服务器，虚拟机和网络设备，采集百万级监控指标。支持分布式架构，通过统一的 Web 界面监控自动监控大型动态环境。
